关闭自动重启（基于bazeport项目）

```c++
setprop persist.bazeport.stub.no_reboot 1
```



唤出设置

```
am start com.android.tv.settings/.MainSettings
```



查看当前的active

```
dumpsys window | grep mCurrentFocus
```



控制台模拟遥控器

```
input keyevent xx

input keyevent 19：向上
input keyevent 20：向下
input keyevent 21：向左
input keyevent 22：向右
input keyevent 23：确认（回车）
input keyevent 4 ：bakc
input keyevent KEYCODE_DEL 删除
```



输入文本

```
input text 文本
```
